# **App Name**: TableBites

## Core Features:

- Menu Browsing: Browse restaurant menu categories such as Starters, Main Course, and Desserts.
- Real-time Order Tracking: Provide real-time updates to customers on their order status: Preparing, Cooking, Served.
- Admin Order Management: Enable the admin/waiter to receive orders with table numbers, and update the order status.
- Cart Management: Add, remove, and update quantities of items in the cart, displayed in a sticky sidebar.
- Dynamic Recommendation Tool: Based on order history and real time menu status, the AI suggests dishes to add to the cart

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to establish a premium and elegant atmosphere.
- Background color: Off-black (#121212) provides a sleek dark theme, enhancing visual appeal.
- Accent color: Vibrant fuchsia (#FF00FF) highlights interactive elements and CTAs, ensuring visibility.
- Body text: 'PT Sans', a humanist sans-serif, for the primary text.
- Headline font: 'Playfair', a modern serif font, bringing an elegant and fashionable style to headlines. If longer text is anticipated, use this for headlines and 'PT Sans' for body
- Use minimalistic icons for menu categories and order statuses to enhance clarity.
- Employ a grid-based layout with consistent spacing and rounded corners for a professional look.
- Incorporate subtle animations for cart updates and order status transitions.
